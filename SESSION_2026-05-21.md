### 5. Bug Fix — Field Alias สำหรับ `branchFrm` (commit `eda3289`)

**ปัญหา:** Mother-PC ส่ง payload ที่มี field `branchFrm`, `branchTo`, `whFrm` ฯลฯ แต่ server `pick()` ไม่รู้จัก field เหล่านี้ → validation error → HTTP 500

**แก้ไขใน `normalizeTransferHeaderRecord`:**
```javascript
branchCode: pick(record, ["FTBchCode", "branchCode", "branchFrm"]),   // เพิ่ม branchFrm
branchCodeTo: pick(record, ["FTBchCodeTo", "branchCodeTo", "branchTo"]), // เพิ่ม branchTo
warehouseCode: pick(record, ["FTWahCode", "warehouseCode", "whFrm"]),   // เพิ่ม whFrm
```

ทำให้รองรับ payload จาก mother-PC ได้โดยไม่ต้องเปลี่ยน sync agent
